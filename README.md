# Atividade Prática – Layout Web Responsivo

## Objetivo da atividade
Este projeto consiste no desenvolvimento de uma página web simples chamada Estuda+, utilizando HTML5 e CSS3. O objetivo é apresentar algumas dicas básicas para melhorar os estudos, por meio de uma interface organizada contendo cabeçalho, menu de navegação, área principal com cards informativos e rodapé.

Além disso, o projeto tem como finalidade praticar conceitos de estruturação de páginas web, estilização com CSS, Flexbox, Grid Layout e responsividade.

## Estratégia de responsividade adotada
A estratégia utilizada foi Mobile First, onde o layout inicial foi desenvolvido pensando em dispositivos móveis e, posteriormente, adaptado para telas maiores utilizando media queries.

## Breakpoints utilizados
- 600px (Tablet)
    Ajusta o cabeçalho para organizar o logo e o menu em linha, melhorando a navegação em telas médias.
- 900px (Desktop)
    Os cards informativos passam a ser exibidos em duas colunas, utilizando CSS Grid, formando uma grade 2x2.
